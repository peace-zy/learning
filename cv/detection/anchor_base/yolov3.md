# Motivation
RCNN系列使用RPN生成候选区域，然后对所生成的后续区域进行分类得到正例和并对正例位置回归，接着对正例区域进行剪切并提取特征，最后基于提取的特征再次进行类别预测和二次位置回归。虽然精度较高，但整体流程较复杂，导致速度较慢、难于优化。
